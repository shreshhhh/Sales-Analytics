# End-to-End Sales Analytics Dashboard

## Overview

This project demonstrates an end-to-end sales analytics workflow using Python, SQL, and Power BI. The objective is to transform raw retail transaction data into actionable business insights through data cleaning, exploratory analysis, KPI monitoring, SQL-based analytics, and interactive dashboards.

---

## Business Problem

Retail organizations generate thousands of transactions every day. Decision-makers require timely insights into sales performance, customer behavior, profitability, and regional trends.

This project builds an analytics pipeline that answers key business questions and provides an executive dashboard for performance monitoring.

---

## Objectives

- Clean and preprocess raw sales data
- Perform exploratory data analysis (EDA)
- Calculate business KPIs
- Analyze sales using SQL
- Build an interactive Power BI dashboard
- Generate actionable business insights

## Key Business Insights

- Technology category generated the highest profit.
- West region contributed the largest share of revenue.
- Furniture showed lower profit margins despite high sales.
- Sales peaked during the holiday season.
- A small percentage of customers contributed a significant portion of total revenue.
